
# House of Spectacular Opulence
### Just X Exclusive

A premium luxury fragrance e-commerce prototype built using HTML, CSS, and JavaScript.

---

## 🌟 Overview

House of Spectacular Opulence is a black & gold luxury-themed boutique website
designed to showcase and sell:

- Linen & Air Spritz
- Scented Candles
- Luxury Diffusers

This version includes a fully functional front-end cart system and WhatsApp order integration.

---

## 🚀 Features

✔ Luxury black & gold UI  
✔ Multi-product cart system  
✔ Automatic quantity increase  
✔ Subtotal calculation  
✔ R80 nationwide delivery fee  
✔ Total calculation  
✔ Customer checkout form  
✔ WhatsApp order submission  
✔ Mobile responsive layout  

---

## 🛒 How Ordering Works

1. Customer adds products to cart.
2. Customer views cart.
3. Subtotal is calculated automatically.
4. Delivery fee (R80) is added.
5. Customer fills in:
   - Full Name
   - Phone Number
   - Email
   - Delivery Address
6. Order is sent directly via WhatsApp.
